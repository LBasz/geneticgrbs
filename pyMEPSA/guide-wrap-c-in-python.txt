Come funziona:
1) Creare la libreria in c clib.c
2) Creare il file python clib.py. Dentro al codice py: l'istruzione ctypes.CDLL(path) apre
la libreria .c desiderata. E' necessario che il path sia un path assoluto, altrimenti si bugga
e non riesce ad aprire il file
3) Compilare il codice in maniera position independent (PIC) con il seguente comando:
	gcc -c -Wall -Wno-error -fpic clib.c
4) Creare una shared library dal file compilato:
	gcc -shared -o clib.so clib.o
5) Eseguire lo script di Python

Vedi dentro i clib.c e clib.py per qualche commento sul codice usato (ho copiaincollato dal tutorial).
